This is Earthquakes by Eli Cohavi.

This is a project using THREE.js and is hosted on web server 'edplay.org/eliasc/convectioncurrents' using FTP Cyberduck. 'node-modules' cannot be uploaded. Libraries used are in source code.

TO USE: use terminal and type 'npm i vite' to install vite node. Then type 'npx vite' to host a local version through browser.
